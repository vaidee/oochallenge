Below are list of features that are implemented in this web application.

  • User can view the list of videos.
  • custom ooyala player v3 is embedded .
  • Click on the asset will load the asset in player & stream the content..
  • User can edit the title of a asset..
  • Below the video , user can see the metadata associated with the video..
  • User can read the metadata and play from that particular frame by a click on play.
  • By player pause user can add a metadata about the current frame(in progress) .
  • Code cleanup & testing (in progress)
